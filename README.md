**Функция в программировании** – это именованный блок кода, который можно вызвать в другой части программы, обратившись по имени. При вызове происходит поочередное исполнение инструкций, находящихся в теле функции. Для объявления функций в Python используется оператор def. Функция состоит из заголовка и тела. В заголовке после имени в круглых скобках перечисляются параметры функции, а в конце заголовка обязательно ставится двоеточие. Чтобы использовать функцию, ее нужно вызвать. Параметрами называют переменные, которые используются при создании функции. Аргументы – это значения, которые передаются функции во время ее вызова для выполнения определенных операций. 


def sum_f(a,b): # a и b – параметры
    print(a+b)


Функция может возвращать результат в основную программу для дальнейшей работы. Для этого используется оператор return, после которого указывается возвращаемое значение. return завершает выполнение функции, поэтому чаще всего пишется в конце функции или в специальном условии. 


def sum_f(a,b, c=50):
    print(f'a={a},b={b},c={c}')
    return a + b * c


Функция является сложной структурой, которая может содержать собственные переменные и функции внутри блока кода. Когда мы объявляем переменные внутри функции, они имеют локальную область видимости. Мы не можем получить к ним доступ вне функции. Такие типы переменных называются локальными переменными.


Переменные, созданные в основном коде программы, находятся в глобальной области видимости. Это означает, что к ним можно получить доступ в любой части программы. В следующем примере глобальная переменная используется в локальной области видимости.
